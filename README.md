## ✨ 주요 기능 (Key Features)

### 1. 🎨 프리미엄 카드 UI (Premium Card Design)
- 직관적인 **대형 카드 버튼**으로 누구나 쉽게 사용할 수 있습니다.
- 복잡한 메뉴 대신 **아이콘과 설명** 중심의 디자인을 채택했습니다.
- **Glassmorphism** 스타일과 부드러운 호버 효과로 고급스러운 경험을 제공합니다.

### 2. ⚡ 강력한 압축 성능
- **다양한 포맷 지원**: ZIP, 7Z 압축 및 ZIP, 7Z, RAR 등 대부분의 포맷 해제 지원.
- **SFX (EXE) 생성**: 별도의 프로그램 없이도 실행만 하면 풀리는 **자동 압축 풀림 파일(.exe)**을 만들 수 있습니다.
- **압축률 조절**: 작업 속도 우선(1단계)부터 최대 압축(9단계)까지 선택 가능.

### 3. 🛡 보안 및 안정성
- **AES-256 암호화**: 은행 보안 수준의 강력한 비밀번호 잠금을 지원합니다.
- **시스템 권한 충돌 방지**: Program Files 등 시스템 폴더 접근 시 발생하는 튕김 현상을 완벽하게 해결했습니다 (AppData 기반 설계).
- **무결성 검사**: 파일의 변조 여부(SHA-256)를 즉시 확인할 수 있습니다.

### 4. 📂 편리한 탐색기 연동
- **탐색기 우클릭 메뉴**: "ParkZIP으로 열기", "여기에 압축 풀기"를 지원합니다.
- **즉시 실행**: 더블클릭 시 불필요한 팝업 없이 바로 파일 목록을 보여줍니다.
- **스마트 드래그&드롭**: 파일을 끌어다 놓기만 하면 자동으로 목록에 추가됩니다.

- 자유롭게 사용할 수 있는 프리웨어입니다.
